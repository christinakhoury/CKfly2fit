# CKfly2fit – Aerial Arts Website

## Description
CKfly2fit is a multi-page website dedicated to Fly2Fit, an aerial arts and fitness studio. This site provides visitors with information about the studio, quizzes to test their knowledge about aerial arts, class schedules, and a contact form for reservations. The website is designed to be interactive, user-friendly, and informative, serving both as a promotional platform and a functional tool for users to engage with the business.

---

## Features
- Homepage: Introduction to Fly2Fit with an overview of the studio’s mission and offerings.
- CV Page: Showcases the tutor’s qualifications, experience, and expertise in aerial arts.
- Quiz Page: Allows users to test their knowledge about aerial arts and the Fly2Fit business.
- Schedule Page: Displays available class times for users interested in joining sessions.
- Contact Page: Lets users enter their name and information to reserve classes. Once submitted, a confirmation message displays the user’s info.
- Responsive Design: Clean layout styled using CSS for a professional and visually appealing look.
- PHP Backend: Handles the contact form submission and displays user confirmation messages dynamically.

---

## Technologies Used
- HTML5 – for website structure
- CSS3 – for styling and responsive layout
- PHP – for backend form handling
- Optional: can be deployed on local server (XAMPP/WAMP) or live hosting

## How to Run
- Clone the repository:
- git clone https://github.com/christinakhoury/CKfly2fit.git
- Open the project in a local server environment like XAMPP or WAMP to handle PHP.
- Navigate to index.html in your browser to start browsing the website.
- Use the Contact page to submit a reservation; the PHP script will display the entered info as a confirmation.
